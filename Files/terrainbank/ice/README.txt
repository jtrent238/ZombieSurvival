This terrain set for Reloaded simulates an arctic region with slight hints of vegetation.
It is suited for use with HIGH Ambience settings, it is recommended to have the ambience
sliders (Power and colors) around 80 - 90.

Created for FPSC Reloaded - use in other engines or as texture pack for other engines prohibited.
